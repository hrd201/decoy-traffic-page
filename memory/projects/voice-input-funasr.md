# 语音输入方案 - FunASR (讨论记录)

## 需求目标

像输入法一样连续性输入：点击开始 → 实时连续识别 → 自动输入到光标位置 → 点击结束

## 方案选择

选择 FunASR 而非 Moonshine

- FunASR 内置 VAD + ASR + 标点处理，一条龙服务
- FunASR 中文效果极佳（阿里开源）

## 推荐模型

SenseVoice-Small (~400MB) - 推荐，M1 完全流畅运行
Paraformer-zh (~500MB) - 准确率更高

## 工作流程

1. 点击开始（快捷键/托盘按钮）
2. 麦克风常开 → VAD 检测语音开始/结束
3. FunASR 实时识别 → 自动添加标点
4. 检测到完整句子后 → 自动粘贴到光标位置
5. 点击结束 → 停止
