# 同花顺基金分析 Skill

通过同花顺网站（fund.10jqka.com.cn）分析基金详情，包括基本信息、业绩表现、重仓持股、投资组合等。

## 使用场景
- 查询基金净值、涨跌幅
- 分析基金业绩表现（近1周/1月/3月/6月/1年/2年/3年）
- 查看基金重仓持股
- 查看资产配置、行业配置
- 对比两只基金

## 前置要求
1. OpenClaw 浏览器已配置好 openclaw profile
2. 浏览器服务正常运行

## 操作流程

### 1. 打开基金页面
使用浏览器打开：`https://fund.10jqka.com.cn/{基金代码}/`

示例：
```
基金代码：588000
URL：https://fund.10jqka.com.cn/588000/
```

### 2. 获取页面信息
使用 browser 工具的 snapshot 获取页面内容，可提取：
- 单位净值、日增长率
- 基金规模、成立时间
- 基金经理信息
- 重仓持股
- 业绩表现

### 3. 点击分页获取更多信息
页面分页包括：
- 基金档案（默认）
- 基金简介
- 历史净值
- 投资组合
- 基金公告
- 财务数据

点击对应链接获取详情：
```python
# 点击投资组合
click("投资组合")

# 点击基金公告  
click("基金公告")

# 点击财务数据
click("财务数据")
```

### 4. 分析关键数据

#### 基本信息
- 基金代码、名称
- 基金规模
- 基金经理
- 管理人

#### 业绩表现
提取表格数据：
| 时间段 | 收益率 | 同类平均 | 沪深300 |
|--------|--------|----------|---------|
| 近1周 | +x% | +x% | +x% |
| 近1月 | +x% | +x% | +x% |

#### 重仓持股
| 排名 | 股票名称 | 占净值比 |
|------|----------|----------|
| 1 | xxx | x.xx% |

### 5. 基金对比
同时打开两个基金页面进行对比：
- 588000：https://fund.10jqka.com.cn/588000/
- 159819：https://fund.10jqka.com.cn/159819/

## 输出格式

### 简报格式
```
# 📊 {基金名称}

**基本信息**：
- 代码：{代码}
- 规模：{规模}
- 经理：{经理}

**业绩表现**：
- 近1年：{涨幅}%
- 近2年：{涨幅}%

**重仓股**：{股票1}({占比}%)、{股票2}({占比}%)
```

### 详细报告格式
包括：
1. 基本档案
2. 费率结构
3. 资产配置
4. 行业配置
5. 重仓持股
6. 历史净值
7. 业绩表现
8. 基金经理
9. 投资建议

## 注意事项
1. 页面数据可能需要等待加载
2. 不同基金的页面结构可能略有差异
3. 历史净值数据更新可能有延迟
4. 基金业绩仅供参考，投资有风险
