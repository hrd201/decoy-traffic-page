# Humanizer - 去除 AI 写作痕迹

让文字读起来更像人写的，更加自然。

## 功能

检测并修复 AI 写作特征：
- 夸张的符号主义
-  promotional 语言
- 过多的 -ing 分析
- 模糊的归属
- 破折号滥用
- 三元规则
- AI 词汇
- 负面平行结构
- 过多的连接词

## 使用方法

### 手动处理

直接把需要 humanize 的文字发给我，我会帮你改写得更自然。

### 自动化处理

```bash
# 使用 Python 实现简单版本
python3 << 'EOF'
import re

def humanize(text):
    # 去除过度使用的破折号
    text = re.sub(r'—+', '—', text)
    
    # 去除夸张的词汇
    replacements = {
        "非常": "很",
        "极其": "很",
        "相当": "比较",
        "尤为重要": "重要",
        "深刻理解": "了解",
        "全面": "完整",
        "系统性地": "系统地",
    }
    for old, new in replacements.items():
        text = text.replace(old, new)
    
    # 减少连接词
    connectives = ["此外", "另外", "同时", "并且", "因此"]
    for c in connectives:
        if text.count(c) > 1:
            text = text.replace(c, "", 1)
    
    return text

# 测试
test = "这个工具非常重要，可以帮助你非常全面地理解问题的各个方面。此外，它还能..."
print(humanize(test))
EOF
```

## AI 写作常见特征

| 特征 | 示例 | 修正 |
|------|------|------|
| 破折号滥用 | 使用过多 — | 减少使用 |
| 夸张词汇 | 极其、非常、相当 | 替换为普通词 |
| 三元结构 | "第一、第二、第三" | 减少为两点 |
| 连接词过多 | 此外、另外、同时 | 删除部分 |
| 模糊归属 | "研究表明"、"数据显示" | 具体说明 |

## 提示词技巧

让 AI 回复更自然：
- 避免 "首先、其次、最后"
- 减少 "非常"、"极其"
- 少用 "同时"、"此外"
- 去掉车轱辘话

## 集成

当用户要求"humanize"或"让这段话更自然"时，使用此技能。
