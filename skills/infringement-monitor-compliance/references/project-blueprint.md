# Project Blueprint (Compliant)

## Data Inputs
- Official APIs / partner feeds
- Authorized platform exports
- Internal reports / user complaints
- Manual review captures

## Risk Features
- Keyword hit and misspelling hit
- Image/logo similarity
- Seller/account linkage
- Abnormal low price vs official baseline

## Evidence Package
- URL + account + publish time
- Screenshot with visible timestamp
- Relevant text snippet
- Hash/checksum of captured file
- Collector and collection time

## Escalation
- P2: monitor
- P1: legal review
- P0: immediate takedown/legal action
