# 搜索工具 Skill

配置 OpenClaw 使用的搜索引擎。

## 可用搜索引擎

| 引擎 | URL | 说明 |
|------|-----|------|
| **Google** | https://www.google.com/search?q= | 国际主流搜索引擎 |
| 夸克 | https://quark.sm.cn/search?query= | 阿里系，简洁AI增强 |
| 纳米AI | https://www.n.cn/search?query= | 百度AI搜索新产品 |
| 必应 | https://www.bing.com/search?q= | 国际引擎 |

## 使用方法

直接告诉我搜索内容，我会使用浏览器打开 Google 进行搜索。

也可以指定搜索引擎："用夸克搜索 XXX" 或 "用必应搜索 XXX"。

## 当前配置

- **默认搜索：Google**（通过浏览器搜索）
- 可用备选：夸克、纳米AI、必应

## 注意事项

- Google 搜索需要代理才能在中国大陆使用
- 如果代理不可用，会自动切换到国内搜索引擎
