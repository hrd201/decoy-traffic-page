# Homepage Structure (Recommended)

## 1. Hero
- One-sentence value proposition
- Primary CTA + secondary CTA

## 2. Core Sections
- About / Brand story
- Key offerings (3-6 cards)
- Recent updates (latest 3)
- Trust signals (cases, testimonials, metrics)

## 3. Footer
- Contact
- Copyright
- Legal links

## Content Quality Checklist
- Headline <= 18 Chinese chars when possible
- Every section answers one user question
- Avoid filler and repeated claims
- Mobile first: short paragraphs, bullet-heavy
